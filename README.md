# neuropixel_py

To concatentate neuropixel data, run the following code:

```python
#Concatenate all subfolders found in the data directory
python concatenate_data.py -i D:/Main/Data/File 

#Concatenate only the datasets indicated that are found in the data directory
python concatenate_data.py -i D:/Main/Data/File -d 0 2 3 
```
